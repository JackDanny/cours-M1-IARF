CONTENTS
========

This package contains the NetLogo GoGo Board extension.

INSTRUCTIONS
============

The GoGo board extension depends upon the RXTX package for
communicating with the GoGo Boards via your computers serial port.  To
install the RXTX package, available at www.rxtx.org.  The GoGo
extension requires version 2.1 or greater, and uses the gnu.io package
(not the javax.comm package).  The RXTXcomm.jar file needs to be in
your CLASSPATH, or in the extensions subdir.

For models that use this extension, see Code Examples/GoGo
in the models library.
