CONTENTS
========

This package contains the NetLogo matrix extension.

INSTRUCTIONS
============

The matrix extension is pre-installed in NetLogo. For 
instructions on using matrices in your model, or for more 
information about NetLogo extensions, see the NetLogo 
User Manual.
